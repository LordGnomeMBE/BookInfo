deamazonator
Easily find a place to buy a book that isn't amazon!
There are many reasons not to buy things from amazon, from ethical issues to them not actually being the cheapest option. I built this site to make it easy to find an alternative.

Important Notes - please read before continuing

This site does work, but I'm still developing it. Please be patient if it doesn't work for some reason.

As booksprice.co.uk does not have https, the first time you click the comparison link, your browser will warn you that it's not secure. Once you've gone past this, it should not appear again.
Currently the site only works for links that contain the amazon.co.uk/Title-Of-Book/dp/83290582305/ref=trackers-from-search-or-wherever-you-arrived-from part. It doesn't work with clean URLs.

If you need to input a new URL, reload the page.

I'm very much an amateur - I never studied computing beyond Year 9 (non Brits, you can look it up). I built this to practice and I update it in my free time. If anyone with more skills than me has any improvements, please contact me.

If you run into problems

Make sure the link is for a physical book, not an eBook or even not a book at all.
Check the URL includes the ref= part.

Check you've typed the URL correctly.
